# Análisis de Rutas Urbanas con OpenStreetMap (Salta, Argentina)

Este proyecto analiza la red vial de la ciudad de Salta, Argentina, utilizando las librerías OSMnx, NetworkX y Folium.

## Objetivos

- Obtener y visualizar el grafo urbano a partir de los datos de OpenStreetMap.
- Convertir el grafo en GeoDataFrames para facilitar el análisis.
- Aplicar y comparar distintos algoritmos de búsqueda: BFS, DFS y A*.
- Visualizar las rutas óptimas mediante Folium y OSMnx.

## Tecnologías

- Python
- OSMnx
- NetworkX
- Folium
- Pandas / GeoPandas
- Matplotlib

## Resultados

Comparación del rendimiento de los algoritmos en términos de tiempo de ejecución, longitud del recorrido y cantidad de nodos explorados.
Incluye representaciones gráficas de cada ruta calculada.

# Urban Route Analysis with OpenStreetMap (Salta, Argentina)

This project explores the road network of Salta, Argentina using **OSMnx**, **NetworkX**, and **Folium**.

## Objectives
- Obtain and visualize the city graph from OpenStreetMap data.
- Convert the graph into GeoDataFrames for analysis.
- Apply and compare different search algorithms: **BFS**, **DFS**, and **A***.
- Visualize optimal routes using Folium and OSMnx.

## Technologies
- Python
- OSMnx
- NetworkX
- Folium
- Pandas / GeoPandas
- Matplotlib

## Results
Comparison of algorithm performance in terms of time, path length, and explored nodes.  
Includes graphical representation of each computed route.
